template<typename Tag>
struct PDFxTMD::DefaultPDFImplementations< Tag >

Tag types for PDF implementations.

These tag types are used to differentiate between different PDF implementations:

  • TMDPDFTag: Used for Transverse Momentum Dependent PDFs
  • CollinearPDFTag: Used for standard collinear PDFs

To learn more about the different PDF implementations, please refer to the factory classes in the Factory.h file.
