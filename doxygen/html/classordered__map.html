Detailed Description

template<typename Key, typename Value, typename IgnoredCompare = std::less<Key>, typename Allocator = std::allocator<std::pair<const Key, Value>>>
class ordered_map< Key, Value, IgnoredCompare, Allocator >

A minimal map-like container which preserves insertion order.

ValueA type for values.
